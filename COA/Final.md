# Von Newmann Architecture (Stored Memory Program)

Main Memory mein hum Data , Programs Store krte hain. Data is like variables.

CPU -> Components :

ALU : Arithmetic and Logic Unit : Performs arithmetic and logical operations.

Registers: Fastest Memory , Stores Temporary data, Smallest size , sequence of flip flops

Registers attached to CPU inspite having Main Memory : Speed : Burden ko kam krte hain

Control Unit : Timing Signal , Control Signal

Timing Signal : Konsi instruction pehle execute hogi aur konsi baad.

Control Signal : Read Write operations ko operate krne ke liye Control signal use krte hain in registers

# Basic Units of Computer

Control Unit : Kisko kya aur kab krna hai. Jo Control aur timing signals hote hain vo yeah hi de raha hain

Memory : Data , Program

Bus : Baki sabhi components ko connect kr ke rkhta hai, inke beech ka flow bana ke rakhta hain
